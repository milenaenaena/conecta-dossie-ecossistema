# Dossiê do Ecossistema Proptech

Levantamento do ecossistema de startups proptech / imobiliário brasileiro, mantido pelo time de Marketing da **Conecta**.

📄 **Versão navegável:** veja o dossiê publicado via GitHub Pages (link na seção *About* do repositório, à direita).

O conteúdo fica em [`index.html`](index.html) — um arquivo único, sem dependências, que pode ser aberto direto no navegador.

## Atualizações

O dossiê é revisado periodicamente. Sugestões de inclusão ou correção de startups: fale com o time de Marketing.
